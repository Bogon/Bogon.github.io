在Swift中创建自定义集合

        在 Swift 中创建对象或值的集合时，我们通常使用标准库提供的数据结构-例如 Array ，Dictionary 和 Set 。 尽管这三个案例涵盖了大多数用例，但有时创建自定义包装器集合可以使你使代码更可预测，并且更不易出错。

        此次，让我们看看作为应用程序开发人员的我们如何在 Swift 中定义此类自定义集合，以及结合枚举的强大功能，如何让我们为自己创建一些非常漂亮的 API 。
