## Xcode 经常使用到的快捷键

+ 格式化代码
```
control + i
```

<!--more-->

+ 折叠/展开代码块
```Bash
局部折叠（折叠一个函数）: Command+Option+Left/Right
全局折叠（折叠当前文件下的全部函数）：Shift+Command+Option+Left/Right
折叠注释块：（/* */之间的文字）： Ctrl+Shift+Command+Left/Right
折叠当前文件的所有方法：command＋option＋shift＋Left/Right
打开左侧面板： command+0/1
```
